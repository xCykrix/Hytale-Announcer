## Announcer

Welcome to Announcer! This is a server-side based Hytale Mod that enables the ability to send general announcements, titles, display a MOTD on play join, and more to come!

## Installation

1. Place `Announcer.jar` into the `mods` folder in Hytale.
2. Start the Server.
3. Locate `github.xCykrix_Announcer/config.json` to update the default messages on join.

## Commands

/title "Title Test" "Subtitle Test" 10 5 5
- Title
- Subtitle
- Duration
- Fade-in
- Fade-out

/announce "Some long message to display to users."
- Chat Message
